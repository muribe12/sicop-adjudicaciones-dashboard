# SICOP Adjudicaciones Dashboard

Interactive dashboard analyzing procurement (adjudicaciones) data from Costa Rica's SICOP open data portal.

## Institutions covered
- Municipalidad de San José (3014042058)
- Municipalidad de Montes de Oca (3014042053)

## Features
- Top proveedores by awarded amount
- Top 50 adjudicaciones by amount
- Temporal and type/modality analysis
- Spending concentration analysis
- Statistical anomaly detection (2σ)
- Full detail table with CSV exports
- Filters: date range, procedure type, proveedor, status

## Data source
[SICOP Open Data Portal](https://www.sicop.go.cr)
